Style Czcionek

Styl 1: Klasyczny

Przykładowy tekst w stylu klasycznym, który prezentuje prosty wygląd bez dodatkowych dekoracji.

.font1 { font-family: 'Arial', sans-serif; font-size: 16px; font-weight: normal; font-style: normal; text-align: left; text-transform: none; text-decoration: none; letter-spacing: normal; line-height: 1.5; }

Styl 2: Wyróżniający

Przykładowy tekst wyróżniający się dużymi literami i ozdobnymi elementami, które przyciągają uwagę.

.font2 { font-family: 'Georgia', serif; font-size: 18px; font-weight: bold; font-style: italic; text-align: center; text-transform: uppercase; text-decoration: underline; letter-spacing: 1px; line-height: 1.8; }

Styl 3: Monospace

tekst w stylu monospace, który przypomina edytor kodu, często używany w technicznych projektach.

.font3 { font-family: 'Courier New', monospace; font-size: 14px; font-weight: lighter; font-style: oblique; text-align: right; text-transform: lowercase; text-decoration: line-through; letter-spacing: 2px; line-height: 1.3; }

Styl 4: Elegancki

Ten styl prezentuje elegancki krój z wyróżniającym efektem i większą interlinią dla lepszej czytelności.

.font4 { font-family: 'Times New Roman', Times, serif; font-size: 20px; font-weight: bolder; font-style: normal; text-align: justify; text-transform: capitalize; text-decoration: overline; letter-spacing: 1.5px; line-height: 1.6; }

Styl 5: Nowoczesny

Tekst w nowoczesnym stylu z prostym krojem pisma i dużymi literami, idealny do nagłówków.

.font5 { font-family: 'Verdana', sans-serif; font-size: 12px; font-weight: 600; font-style: normal; text-align: left; text-transform: uppercase; text-decoration: none; letter-spacing: 0.5px; line-height: 1.4; }
